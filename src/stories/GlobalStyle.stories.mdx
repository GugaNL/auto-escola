# Global Style

css resets and base stylesheet definition.

```jsx
const App = () => {
  ;<>
    <GlobalStyle />
    <OtherComponent />
  </>
}
```

### Typography

Basic global typography.

### Headings

All html heading, `<h1>` through `<h6>`, are available:

<Preview>
  <Story name="Heading">

  </Story>
</Preview>
